# FXScript for VS Code

VS Code extension for FXScript, an assembly-like scripting language.

## Features

- Syntax highlighting for `.fx` files.
- Workspace-wide symbol indexing (Labels, Variables, Constants, Macros).
- Go to Definition and Find References.
- Code completion for built-in and custom commands.
- Basic validation and error checking.

## Usage

The extension activates automatically when you open an `.fx` file.
Custom commands and identifiers are loaded from `commands.json` in your workspace root.
